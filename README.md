# LLM自作入門

## 2章　テキストデータの準備
### 目的
トークナイザーの実装を通じて、入力テキストをトークン化し、さらにトークンIDに変換する方法を学習  
また、トークンIDからトークンを復元する逆変換についても学ぶ

#### トークナイザーの実装
##### V1の実装
vocab をプロパティに持ち、encode, decode メソッドを実装した simple tokenizer を実装  
テキスト encode の際に、vocab に存在しない場合はエラーとなる  
--> 語彙の豊富さを学習により獲得する必要性がわかる  

未知のワードや文章の境界を示すキーワードの導入が必要
- |unk| *unknownからきてる?
- |endoftext| *end of text

### 用語集
トークン：
    LLMに認識させるテキストの単位。rawデータからトークナイザーが適した形でパース
    トークンはその後、トークンIDという整数値に置き換えられる
トークンID：
    個々のトークンを一意の整数値に変換したもの。
